Cloud for Save

欢迎使用 Cloud for Save

Cloud for Save 是一款现代化、模块化的云存储解决方案，基于 Go 语言开发，支持本地和 S3 兼容存储，具备灵活的权限管理、数据监控、预处理、分享与审计等功能。

  • 多存储后端支持（本地/S3/MinIO）
  • 灵活的配置与权限管理
  • Web 管理界面，实时配置与监控
  • 数据预处理与事件回调
  • 安全的链接分享与访问审计

项目简介

本项目旨在为企业和个人提供高效、安全、可扩展的数据存储与管理能力。支持通过 Web 控制台动态调整服务配置，所有配置变更均可实时保存并持久化，系统自带默认配置备份，便于一键恢复。
